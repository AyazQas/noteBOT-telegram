Привет! Это инструкция пользования ботом.


Инструкция для новичков

1 шаг. Установи Python если он у тебя еще не установлен (https://www.python.org/downloads/)
2 шаг. Открой файл config.py и впиши туда свой BOT_TOKEN (Его можно получить в BotFather (https://t.me/BotFather))
3 шаг. Запусти install.bat (Дождись закрытия командой строки)
4 шаг. Зайди в папку dict и запусти main.exe


Инструкция для знающих

1 шаг. В любом интерпретаторе запусти config.py, впиши туда бот токен.
2 шаг. Напиши в терминал pip install -r requirements.txt
3 шаг. Запусти main.py

Удачного использования!
