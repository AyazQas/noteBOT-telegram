Hello! This is the bot usage instructions.

Instructions for Beginners

Step 1. Install Python if you haven't already (https://www.python.org/downloads/)
Step 2. Open the config.py file and enter your BOT_TOKEN (You can obtain it from BotFather (https://t.me/BotFather))
Step 3. Run install.bat (Wait for the command prompt to close)
Step 4. Go to the dict folder and run main.exe

Instructions for Experienced Users

Step 1. In any interpreter, run config.py and enter your bot token.
Step 2. In the terminal, type pip install -r requirements.txt
Step 3. Run main.py

Enjoy using it!
